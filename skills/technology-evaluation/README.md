# Technology Evaluation Skill

**Version:** 1.0.0
**Status:** Active
**License:** MIT

## Overview

This skill provides comprehensive guidance for evaluating and selecting technologies. It covers requirements analysis, pros/cons analysis, proof of concepts, technology roadmaps, and technology selection following technology evaluation best practices.

## When to Use

**Use this skill when:**

- Evaluating technology options for projects
- Comparing technology stacks
- Conducting technology assessments
- Making technology selection recommendations
- Creating technology requirements
- Performing proof of concept (PoC)
- Evaluating technology vendors
- Conducting technology feasibility studies
- Creating technology roadmaps
- Evaluating technology risks
- Assessing technology scalability
- Evaluating technology maintainability
- Conducting technology cost analysis
- Evaluating technology community support
- Assessing technology learning curve
- Evaluating technology ecosystem
- Conducting technology compatibility analysis
- Evaluating technology security
- Assessing technology performance
- Evaluating technology integration
- Conducting technology migration analysis
- Evaluating technology future outlook
- Assessing technology vendor stability

## What It Covers

### Evaluation Framework

- **Checklist** - Comprehensive evaluation criteria
- **Matrix** - Technology comparison matrix
- **Scoring** - Weighted scoring system

### Technology Comparison

- **Pros/Cons Analysis** - Detailed comparison
- **Template** - Standardized evaluation
- **Recommendations** - Clear guidance

### Proof of Concept

- **Template** - PoC structure
- **Success Criteria** - Defined metrics
- **Deliverables** - Expected outputs

### Technology Roadmap

- **Current State** - Assessment
- **Target State** - Goals
- **Migration Strategy** - Phased approach
