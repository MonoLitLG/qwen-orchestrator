# Strategic Leadership Skill — Command & Master Orchestrator

## Overview

This skill provides comprehensive guidance for strategic leadership, team management, decision-making frameworks, and organizational strategy. It covers leadership styles, stakeholder alignment, crisis management, and orchestrating complex projects following PMP, CSM, and CSP methodologies.

## When to Use

**Use this skill when:**

- Leading and managing cross-functional teams
- Making high-stakes strategic decisions
- Aligning stakeholders on project vision and goals
- Managing crisis situations and emergencies
- Developing organizational strategy and roadmaps
- Coaching and mentoring senior leaders
- Resolving team conflicts and improving dynamics
- Setting performance metrics and KPIs
- Managing executive-level communications
- Developing leadership pipelines and succession planning
- Implementing organizational change initiatives
- Creating and executing business strategy
- Managing portfolio-level priorities
- Aligning technical decisions with business goals
- Leading agile transformations
- Managing remote and distributed teams
- Building high-performing team cultures
- Developing leadership competencies and frameworks
- Creating decision-making frameworks and processes
- Managing stakeholder expectations and communications
- Leading post-mortems and lessons-learned sessions
- Developing leadership development programs
- Creating strategic alignment matrices

**Do NOT use this skill when:**

- Writing application business logic (use domain-driven skill)
- Designing database schemas (use database-design skill)
- Writing API endpoint specifications (use api-design skill)
- Performing security audits (use security-auditor skill)
- Writing SQL queries (use sql-best-practices skill)
- Designing multi-page website layouts (use design-system skill)
- Analyzing deployment configurations (use deployment skill)
- Writing documentation (use documentation-best-practices skill)

## Leadership Styles

### Situational Leadership Model

```
┌─────────────────────────────────────────────────────────────┐
│                    Situational Leadership                   │
├─────────────────────────────────────────────────────────────┤
│  Directive (Low Support, High Task)                         │
│  └── For: New team members, unclear tasks                   │
│  └── Action: Define roles, set expectations, monitor      │
│                                                              │
│  Coaching (High Support, High Task)                         │
│  └── For: Developing team members                           │
│  └── Action: Explain decisions, seek input, support       │
│                                                              │
│  Supporting (High Support, Low Task)                        │
│  └── For: Competent but unsure team members                 │
│  └── Action: Listen, provide support, encourage           │
│                                                              │
│  Delegating (Low Support, Low Task)                         │
│  └── For: Competent and confident team members              │
│  └── Action: Empower, trust, monitor occasionally         │
└─────────────────────────────────────────────────────────────┘
```

### Decision-Making Model

```
┌─────────────────────────────────────────────────────────────┐
│              Decision-Making Process                        │
├─────────────────────────────────────────────────────────────┤
│  1. Define the Problem (Clear, measurable)                  │
│  2. Gather Information (Data, stakeholder input)            │
│  3. Identify Alternatives (Brainstorm, evaluate)            │
│  4. Evaluate Options (Pros/cons, risk analysis)             │
│  5. Make Decision (Based on criteria, not emotions)         │
│  6. Implement Decision (Action plan, assign ownership)      │
│  7. Review Decision (Monitor, adjust as needed)             │
└─────────────────────────────────────────────────────────────┘
```

## Real-World Impact

**Before this skill:**

- Teams lack direction and alignment
- Decisions made reactively
- Stakeholders frustrated with communication
- Crises cause chaos and delays
- Team conflicts unresolved

**After this skill:**

- Clear strategic direction and alignment
- Decisions made proactively and effectively
- Stakeholders informed and engaged
- Crises managed systematically
- Teams collaborate effectively
